# Português em Foco — B1.2 → B2

App pessoal de aprendizagem de Português Europeu, construída com base no currículo do *Português em Foco 3*.

## Instalar no iPhone
1. Abre a URL do GitHub Pages no Safari
2. Toca em Partilhar → "Adicionar ao ecrã principal"
3. Abre como app a partir do ecrã principal

## Instalar no laptop
1. Abre a URL no Chrome
2. Clica no ícone de instalação na barra de endereço
3. Abre como app standalone

## Funcionalidades
- Exercícios de preenchimento e escolha múltipla (B1.2 → B2)
- Gramática de referência rápida com tabelas
- Flashcards de vocabulário da Unidade 1
- Correção automática de textos escritos (IA)
- Sistema de XP e streak semanal
- Funciona offline
